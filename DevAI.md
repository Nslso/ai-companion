# Тестовое задание: Персонализированный учебный ассистент с памятью

Должность: Middle+ Разработчик AI-Агентов
Стек: Python, LangChain, LangGraph, векторная БД (Chroma/FAISS/Qdrant)

## Цель задания
Разработать диалогового AI-агента, который выступает в роли персонального учебного ассистента. Агент должен запоминать контекст обучения пользователя, адаптироваться под его уровень знаний и помогать в изучении выбранной темы.

Ключевой аспект — реализация долгосрочной памяти, управление сложным состоянием диалога и способность агента делать выводы на основе истории взаимодействий.

## Задание
 
1. Требуется реализовать Learning Companion Agent с полной системой памяти
2. Необхоидмо создать демонстрационный сценарий обучения на конкретной теме
3. Подготовьте визуализацию графа диалога (с помощью LangGraph)
4. Напишите документацию, включая:
   * Описание архитектуры системы памяти
   * Примеры сессий обучения
   * Объяснение логики переключения режимов

## Технические требования:
1. Использовать LangGraph для управления workflow диалога
2. Реализовать сложное состояние диалога с помощью Pydantic
3. Интегрировать векторную базу данных для хранения и поиска по истории взаимодействий
4. Реализовать механизм RAG (Retrieval-Augmented Generation) для извлечения релевантной истории
5. Граф должен содержать ноды: `analyze_context`, `retrieve_memory`, `select_mode`, `generate_response`, `update_memory`

## Критерии оценки
1. Кодовая база должна содержать README.md в котором есть все необходимая информация о сервисе (объем информации определить самостоятельно)
2. Приложение должно запускаться в docker
3. Какие аспекты будут оцениваться:
    3.1 Архитектура системы памяти
    3.2 Управление сложным состоянием
    3.3 Логика принятия решений
    3.4 Качество диалога
    3.5 Масштабируемость

## Бонусные задания
1. Визуализация прогресса: Реализуйте генерацию графиков или диаграмм прогресса обучения
2. Мультимодальность: Добавьте поддержку генерации учебных материалов (диаграммы, схемы через инструменты)
3. Адаптивная сложность: Реализуйте динамическую регулировку сложности контента на основе успешности ответов пользователя
4. Коллаборативная learning: Добавьте режим, где агент и пользователь совместно решают проблему
